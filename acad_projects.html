Unreal Engine - Game Design


I am currently learning how to visualize AI-based learning through Game Design tools such as Unreal Engine. This project greatly furthered my understanding of AI and Game Design.

II.

Checkit - Full Stack Development

As part of a team project, we created an online platform for users to plan their daily activities. The website uses APIs to calculate the best route for the user to take, and also provides a list of activities that the user can do at the destination. This project was built using HTML, CSS, and JavaScript.
